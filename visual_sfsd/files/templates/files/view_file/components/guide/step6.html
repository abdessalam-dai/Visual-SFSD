This little indicator when clicked on is used to display your current file characteristics as :

  • Number Of Blocks represents Nb the current number of blocks.
  • Number Of Elements displays te total number of elements even the ones removed logically.
  • Index Table Size prints the value of the maximum length of the index table.
  • Max. number Of Blocks portrays the maximum number of blocks allowed in the file.
  • Max. number Of Elements portrays the maximum number of elements allowed in the file.
